jPlayer

The goal of this android application was to allow users to browse their music library without having to touch their device. This application uses the devices's proximity sensor (if available) to detect the presence of nearby objects. Whether you are running on a treadmill or driving a car, simply wave your hand over the device's sensor to change songs.

Check out jPlayer's source code on GitHub
