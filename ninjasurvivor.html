Ninja Survivor

This game was created as a way to practice my object oriented concepts in java.
The goal of this game is to survive as long as you can from the other ninjas
There are 3 characters you can choose from. The controls are below:

Arrow Keys - Move
Space Bar - Jump
A - Throw Projectiles
S - Ultimate attack
D - Booster
